# `Postgresql` database basic operations

### How to access postgrad shell?
`psql postgres`

### How to create database?
`CREATE DATABASE db_name;`

### List of Database.
`\l`

### How to drop database ?
`DROP DATABASE db_name;`

### How to switch database?
`\c db_name`


### How to create `User`?
`CREATE USER user_name;`

### How to drop User?
`DROP USER user_name;`

### List of users.
`\du`
